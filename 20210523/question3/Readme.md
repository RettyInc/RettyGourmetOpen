# 店舗の営業時間はあっているか?

飲食店の営業時間は様々で、朝早くからやっているお店もあれば、日付が変わって営業を終了するお店もあるなど様々です。一方で明らかに間違っている情報を表示してはメディアとしての信頼性が失われてしまいます。

架空の 30 店舗の営業時間情報を用意しましたので、それぞれの営業時間が妥当か・誤りがあるかを判断してください。妥当な場合(OK)は 0、誤りがある場合(NG)は 1 として扱い、得られた 30 の数字を左から右へ並べ、2 進数とみなして 10 進数に置き換えると口コミを指す ID が得られます。

## 例)

1. 1 店舗目から順に OK -> OK -> NG -> OK -> NG -> NG -> NG ...と結果が出る
2. 30 の結果を並べると 00101111000110000101001110 となる
3. 2 進数と見直して 10 進数に直すと 123456789 となる ← これが答え

## ルール

- 時間は時・分から構成され、24 時間表記である。
  - 12:34 の形式。
- 時が 1 桁の場合、0 が入っているケースもあるし、ないケースもある。
  - 例 8:00 と 08:00 はどちらも OK
- 分が 1 桁の場合、0 が入っていないケースは NG 扱いとする
  - 例 9:5 という時刻表記はしませんよね? 9:05 でないと NG
- データはCSVで提供します。
  - 1行が1店舗の営業時間を表し、左から営業開始時間1, 営業終了時間1, 営業開始時間2, 営業終了時間2… と並びます。
  - 営業時間がない項目は空になっています。
- 営業時間は開店時間、閉店時間の組み合わせで提供される。
    - 開店時間・閉店時間の一方が欠けているケースがある。
    - 開店時間・閉店時間の組み合わせは複数提供されるケースがある(ランチとディナーで営業時間が別れているなど) - 営業時間が複数ある場合は 5 時に近いものを先に記載する。
- 閉店時間が日付を跨ぐ場合のみ、24 よりも大きい数字で表記する
    - 例 翌朝 5:00 閉店の場合 "〜 29:00" となる
    - ※出題では 30:00 を超える時刻は NG 扱いとします。
- ユーザーに信頼できない印象を与えるケースは NG 扱いとする
    - 開店時間と閉店時間が逆転している
    - 複数の営業時間で重なりがある。
